# Risk

Proyecto de riesgos financieros

## Prerrequisitos
- Python 2.7.13
- virtualenv
- python-pip

## Configuración
```
$ virtualenv .env
$ source .env/bin/activate
```

## Instalación
### Dependencias
```
$ pip install django

```
## Crear BD
```
python manage.py makemigrations
python manage.py migrate
```


## Ejucutar
```
python manage.py runserver
```
